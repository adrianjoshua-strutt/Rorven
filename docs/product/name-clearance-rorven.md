# Preliminary name status: Rorven

Status: provisional development name  
Date: 2026-06-16

## Decision

Rorven may be used as the repository and development name. This document is not a legal trademark opinion and does not claim exclusive rights.

The project intentionally defers formal clearance, domain acquisition, and package-name reservation until they are operationally useful. Before a stable package release or commercial branding, the project must repeat live checks for:

- GitHub organization and repository names,
- npm package and scope,
- PyPI project,
- container registry namespace,
- selected domains,
- relevant exact and similar trademarks.

## Rename readiness

A future rename must not require domain or persistence migrations. Branding is centralized in `.project/identity.yaml` and package metadata. Domain entities, database schemas, event names, API routes, ports, and adapter contracts remain brand-neutral.
